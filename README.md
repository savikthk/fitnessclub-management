# Fitness Club Management System

Система управления фитнес-клубом с полным набором функций для администрирования клиентов, тренеров, абонементов и тренировок.

##  Основные сущности

- **Member** - клиенты фитнес-клуба
- **Trainer** - тренеры
- **Membership** - абонементы клиентов
- **TrainingSession** - персональные тренировки
- **WorkoutClass** - групповые занятия
- **Booking** - бронирования на групповые занятия

## Бизнес-операции

### 1. Запись на персональную тренировку
- Бронирование персональной тренировки с тренером
- Проверка доступности тренера
- Автоматическое управление статусами тренировок

### 2. Автоматическое продление абонемента
- Продление существующего абонемента
- Автоматический расчет новой даты окончания и стоимости
- Создание нового абонемента при продлении

### 3. Генерация финансового отчета
- Анализ доходов за указанный период
- Группировка по типам абонементов
- Статистика по проданным абонементам

### 4. Поиск клиентов с истекающим абонементом
- Автоматическое определение клиентов для уведомлений
- Настройка периода уведомления (по умолчанию 7 дней)
- Детальная информация о оставшихся днях

### 5. Уведомление о пропущенных тренировках
- Автоматическое обнаружение пропущенных тренировок
- Изменение статуса на "NO_SHOW"
- Отчет для последующих действий

## Технологии

- Java 17+
- Spring Boot 3+
- Spring Data JPA
- PostgreSQL
- Maven

## API Endpoints

Полная коллекция запросов доступна в файле `Postman_Collection.json`

### Основные endpoints:
- `GET/POST/PUT/DELETE /api/members` - управление клиентами
- `GET/POST/PUT/DELETE /api/trainers` - управление тренерами
- `GET/POST /api/memberships` - управление абонементами
- `GET/POST /api/training-sessions` - управление тренировками
- `GET/POST /api/workout-classes` - управление групповыми занятиями
- `GET/POST /api/bookings` - управление бронированиями

##  Запуск проекта

1. Убедитесь, что установлены Java 17+ и PostgreSQL
2. Настройте подключение к БД в `application.properties`
3. Запустите приложение:
   ```bash
   ./mvnw spring-boot:run
